Designing Lamps

HCDE533 Assignment 6

Instruction

Please make a lamp! Using your lamp innereds' measurements as a point of departure, make a lamp that can be assembled around the innereds out of 3D printed parts. You should be able to remove the lamp from the innereds again, so you cannot attach the lamp to the innereds with glue, fasteners, adhesive, etc. Imagine your lamp being in a domestic setting---perhaps it will not be wildly shaken, but it should not fall apart with normal use.

Tools

Rhino, Cura, Ender 3D Pro

Process

Inspiration

During my brainstorming, I found these pictures of vases and lamps. I really like these simple and curvy designs and decided to create something similar.

Watch Tutorial Videos

Before moving forward, I wanted to make sure I’m able to make those shapes with Rhino. So I watched several tutorial videos on YouTube to learn about how to use Rhino and Grasshopper. As Grasshopper seemed pretty complicated, I decided to use only Rhino to create a 3D model using some commands that I learned from the videos.
