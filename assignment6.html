Designing Lamps

HCDE533 Assignment 6

Instruction

Please make a lamp! Using your lamp innereds' measurements as a point of departure, make a lamp that can be assembled around the innereds out of 3D printed parts. You should be able to remove the lamp from the innereds again, so you cannot attach the lamp to the innereds with glue, fasteners, adhesive, etc. Imagine your lamp being in a domestic setting---perhaps it will not be wildly shaken, but it should not fall apart with normal use.

Tools

Rhino, Cura, Ender 3D Pro

Process

Inspiration

During my brainstorming, I found these pictures of vases and lamps. I really like these simple and curvy designs and decided to create something similar.

Watch Tutorial Videos

Before moving forward, I wanted to make sure I’m able to make those shapes with Rhino. So I watched several tutorial videos on YouTube to learn about how to use Rhino and Grasshopper. As Grasshopper seemed pretty complicated, I decided to use only Rhino to create a 3D model using some commands that I learned from the videos.

  • Vase Design with Grasshopper
  • Creating a Tower in Grasshopper
  • Rhino Grasshopper Comples Twist
  • Parametric Vase - Grasshopper Tutorial
  • Create Pattern Vase

Take Measurement 1

I measured each part of my lamp innards.

Take Measurement 2

I also mearured it with a light bulb to make sure the lamp innards and a light bulb will be covered completely.

Sketching the idea

After measuring the lamp innards, I sketched out my ideas and decided on the lamp size. During this sketching session, I used Rhino sometimes to make sure that I am able to create the shape.

Framing

Before starting the modeling the shape, I created the frame of my wavy tower. This helped me see the height, and width of the top/bottom opening easily while I’m building the shape.

Create a wavy circle

I first created a little circle and with the command ArrayPolar, Trim, and Join, I made this large wavy circle. This is the shape of the bottom part of the lamp.

Create a wavy path

I used a command Curve and outlined the wavy path from the bottom to the top of the tower.

Create a body

Using the command RailRevolve, I created this body shape - the wavy circle at the bottom traces the wavy path and create this surface.

Twist a body

Using the command Twist, I twisted the body 45 degrees and added a shade on the surface as well.

Give it a width

Using the command OffsetSrf, I added a 2 mm width to the surface.

As I noticed the lamp innards have little slots, I added the —— so that the lamp will stay on the innards tight.

Test print

For the test print, I only needed to make sure the lamp sits on the innards nicely. I printed only the top part of the lamp and put it on the innards. It didn’t fit very tight, but I made sure it was balanced and didn’t tilt when I move it around.

Print

I converted the file to STL and exported it to Cura. I flipped the body to minimize the amount of support I use on Cura. It took around 16 hours to print the entire body.
