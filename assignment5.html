Getting Meshy

HCDE533 Assignment 5

Instruction

Modify the mesh of two STLs to produce a single printable STL. You may use whatever STLs you want, e.g. downloaded from Thingiverse or elsewhere! When exporting the STL from Rhino, it should pass all checks for rapid prototyping.

Tools

Rhino, Cura, Ender 3D Pro

Process

Import STL files

I found these cute bears on Thingiverse and imported the STL files to Rhino (File -> Import)

Check the opject type

After clicking the object, I found the "Properties" of the object on the side bar on the right of the screen. The type of both object I imported were already closed mesh.

Combine 2 object

Since both opjects were closed mesh, I wanted to see if I can combine them without reducing mesh using a command BooleanUnion. The command didn't work.

Reduce mesh

After reducing mesh by 50% three times using a command ReduceMesh, I tried combining the two object again with BooleanUnion. This did not work again.

Troubleshoot

I measured each part of the pieces with calipers. I found some parts were slightly larger than 5 mm while others were slightly smaller than 5 mm. I decided to make the width of slots 1mm larger and the width of box joints 1mm smaller on Rhino. I printed the same pieces and they fit well.

Check with Cura

For the lid design, I put 35mm square over 25mm square to make it fit tight in the box. It didn’t work when I tried printing it without any support (Image1), so I added support on Cura, which was an easy fix. The support came off very easily as well. (Images 2 & 3)
