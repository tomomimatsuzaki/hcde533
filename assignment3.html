Designing with Grasshopper 1 - Square

First, I created this square part with Rhino and Grasshopper. I used Move to center the square body first, which helped me easily place the slots. I assigned a small number for Radius to create round coners. The slots have to stick out of the square to be able to trim with regions. See the full definitions below.

Designing with Grasshopper 2 - Triangle

I used Polygon to create the triangle (Segments=3). For the Polygon, I set Radius 18 as the width of a triangle and Fillet Radius 4 to make round corners. I created the slot on the left first so that I can create the other two slots using Polar Array instead of having to rotate the rectangle. See the full definitions below.

Transfer to Illustrator

Once the design is finalized, export the Rhino file (2D designs) as an AI file. One important tip for this process is to select the right scale when exporting a Rhino file to Illustrator - input "1mm = 0.0394 inch" Then, open the Illustrator file and change the stroke of the outline to 0.01 pt. To print with the laser cutter at The 8, we have to change the the outline color to red for cutting.
