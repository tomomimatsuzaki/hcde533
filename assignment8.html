Propagation Station

One of my ideas for the final project was to create something for plants, but I wanted to make something more intricate than just a plant pot. During my brainstorming session, I came across some images of plant propagation and thought it would be a great idea to create a propagation station. My propagation station is going to be a tool that allow users to propagate many plants at the same time. See the inspo and my sketchese below for more details.

Inspo

Sketches

How it works

The design of the propagation station would look like stairs. Each little plate that hold test tubes will be a flower petal shape and I'm going to lazer cut a acrylic sheet to create this shape. There will be around 5-6 petals and each petal holds one test tube.

Top & bottom

When you see this propagation station from the top, it will look like a flower. The shape of the base is also going to be a flower shape. I will create this base with 3D printer.

Structure

One thing that I need to make sure is that the acrylic flower petal shape plates can hold a test tube without tilting when they're filled with water. Therefore, I decided to create Lego inspired round brick to hold each acrylic plate tight.
